# 🌐 Projeto Site - Curso de Git e GitHub

Este repositório foi criado durante meus estudos de **Git e GitHub**, acompanhando as aulas do Curso em Vídeo.

O objetivo foi aprender na prática como versionar projetos, utilizar o GitHub e publicar um site utilizando o GitHub Pages.

* HTML
* CSS
* Git
* GitHub

### 🔹 Fundamentos

* O que é Git

* O que é GitHub e para que serve
* Evolução do Git e GitHub

## 📈 Minha evolução com Git e GitHub

Antes deste curso, eu já havia realizado algumas publicações no GitHub, porém sem um entendimento aprofundado das ferramentas.

Com o estudo estruturado de Git e GitHub, passei a compreender melhor conceitos como:

* Versionamento de código
* Uso correto de commits
* Organização com branches
* Utilização de issues
* Escrita de documentação com Markdown
* Publicação de projetos com GitHub Pages

Esse projeto representa minha transição de um uso básico para um uso mais consciente e profissional das ferramentas.
